这里主要是以2维数组为例，其他的道理一样。
对于已经指定键值的数组，推荐用foreach:
$arr = array(
    'one'=>array('name'=>'张三','age'=>'23','sex'=>'男'),
    'two'=>array('name'=>'李四','age'=>'43','sex'=>'女'),
    'three'=>array('name'=>'王五','age'=>'32','sex'=>'男'),
    'four'=>array('name'=>'赵六','age'=>'12','sex'=>'女')
	     );
foreach($arr as $k=>$val)
{
   echo $val['name'].$val['age'].$val['sex']."<br>";
}

这里也存在一个问题，如果数组中的元素有很多，如果一个一个的写出键值的方法就不是适用了，可不可再以写一个循环捏。
将上面echo内容换成
foreach($value as $t=>$d)
	 {
    echo $value[$t].'<br>';
     }
虽然实现了任意多个元素随意输出，但是仍然是有问题的，待完善。

对于未指定键值的数组，推荐用for循环：
$arr=array(
array('nihao','beijing','wwwww','eeeeeee'),
array('woshi','dj','sssss','vvvvv'),
array('woai','xwq','ccccc','bbbbbbbbbbb')
);
for ($row = 0; $row < 3; $row++) 
{
  echo "<p><b>Row number $row</b></p>";
  echo "<ul>";
  for ($col = 0; $col < 4; $col++) 
   {
    echo "<li>".$arr[$row][$col]."</li>";
   }
  echo "</ul>";
}
