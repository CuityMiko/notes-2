###nil
golang的nil在概念上和其它语言的null、None、nil、NULL一样，都指代零值或空值。nil是预先说明的标识符，
也即通常意义上的关键字。在golang中，nil只能赋值给指针、channel、func、interface、map或slice类型的变量。
如果未遵循这个规则，则会引发panic。
package main
import "fmt"
func main(){
    var a int
    var b float32
    var c bool====
    var d string
    var e []int
    var f map[string] int
    var g *int
    if nil == e{
        fmt.Print("e is nil \n")
    }
    if nil == f{
        fmt.Print("f is nil \n")
    }
    fmt.Print(a,b,c,d,e,f,g)
}

##自动类型转换
package main
import "fmt"
func main(){
	var b string
	b="Hello world"
	fmt.Print(b)
}
上面的相当于：
package main
import "fmt"
func main(){
	b := "Hello world"
	fmt.Print(b)
}
go语言编译器自动会推断变量b的类型。
